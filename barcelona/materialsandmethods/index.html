USING R STUDIO AND ITS COMMANDS

In this project we used R Studio, an integrated development environment designed for the R programming language. It is commonly used for statistics and data analysis because it allows users to write code, test commands and visualize results in the same workspace. In our research, RStudio helped us process and analyse air quality data in order to generate different graphics that show how pollution changes over time.

Before starting the analysis, we prepared the working environment by installing several libraries with the command install.packages(). The most important ones were tidyverse, which helps organise and transform data, dplyr for manipulating data tables, lares for cleaning and replacing text values, and openair, a package designed for the analysis of atmospheric pollution and wind data. These libraries were loaded using the library() command so their functions could be used later in the code.

After preparing the environment, we imported the dataset using read.csv(). The original data was stored in a wide format, where each hour of the day appeared in a different column. To make the data easier to analyse, we converted it into a long format using the command pivot_longer(). We also removed unnecessary columns so the dataset became simpler and easier to work with.

Another important step was formatting the date and time information. The date and hour columns were combined using mutate() and paste0(). Then the time format was cleaned using replaceall(). After this, the column was converted into a proper date-time format using as.POSIXct(). Finally, the pollutant variable was converted into a factor with as.factor() so that pollutants could be grouped in graphs.

Once the data was prepared, we created visualizations using the openair package. The function timeVariation() shows how pollution changes depending on the hour, day or month. Another function, trendLevel(), was used to analyse the evolution of a specific pollutant such as H2S. These graphics helped us understand pollution patterns and present the results of our research more clearly.
