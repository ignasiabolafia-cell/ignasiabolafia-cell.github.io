PWM: Aconseguim una sortida analògica

PWM significa Pulse With Modulation, en català mosulació de l'amplada del pols

A la següent imatge podem veure que si vull una sortida analògica en ardunio per controlar un LED necesito donar polsos molt ràpidament en milisegons de cinc volts. Aquests polsos són d'ona cuadrada, és a dir, canvien de 0 a 5 volts de forma immediata. Els canvis són tant ràpids que l`'ull humà no el veu. Si fos un motor servo d'un braç robòtic la inèrcia del motor faria que no s'aturi el motor.

Sui dono 5 volts la meitat del temps pujant i baixant de 0 a 5 volts en ones cuadrades continuament sera com si dones 2,5 volts i el motor anira a la meitat de velocitat i el led s'encendra la meitat.

Pots veure una simulació de blonking amb arduino a continuació

En la següent imatge podem veure com funciona internament PWM.

A la imatge de més amun podem veure que amb 0 volts es mante tota l'estona a 0, i 5 volts es mantindra constant

Podem que el 90% de PWM vol dir que esta fent 0-1-0-1-0-1... de forma que l'amplada de 0 volts, és a dir, quan és 0 i es la mateixa que l'amplada de 1, que es 5 volts. Això es un voltatge de 2,5V

Podem que el 90% de PWM vol dir que esta fent 0-0-0-1-0-0-0-1... de forma que l'amplada de 0 volts, és a dir, quan és 0 igual es tres vegades l'amplada de 1 que es 5 volts. Això es un voltatge de 1,25V (o,25*5)

Podem que el 90% de PWM vol dir que esta fent 1-1-1-0-1-1-1-0... de forma que l'amplada de 0 volts, és a dir, quan és 0 es una quarta part de l'amplada de 1, que es 5 volts. Això es un voltatge de 3,75V (0,75*5)

// C++ code
//Al principi sempre hi ha d'haver un comentari explicant que fa el codi següent.
// El primer codi que s'ha d'executar, per asegurar que la placa es reconeguda i el port és reconegut
//Aquest codi es blinking o parpelleig i encen o apaga el led

void setup()
{
  pinMode(13, OUTPUT);
}

void loop()
{
  digitalWrite(13, HIGH);
  delay(1000); // Wait for 1000 millisecond(s)
  digitalWrite(13, LOW);
  delay(1000); // Wait for 1000 millisecond(s)
}
